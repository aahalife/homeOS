# Note to Actions Skill

Transform information into action through atomic habit extraction and implementation planning.

## Philosophy

This skill applies behavior change science to extract actionable habits from any content:

1. **Atomic**: Break down to smallest possible action
2. **Obvious**: Create clear cues/triggers
3. **Attractive**: Link to existing desires
4. **Easy**: Reduce friction to minimum
5. **Satisfying**: Build in immediate rewards

## When to Use

- User shares an article or link
- User shares a YouTube video
- User mentions a book or podcast
- User asks "how do I apply this?"
- User wants to turn an idea into action
- User says "I want to start doing X"
